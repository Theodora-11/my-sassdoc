    flexCenter

    @mixin flexCenter($direction: row, $justify: center, $align: center) { ... }

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $direction

    The direction of the flex container (row, column, row-reverse, column-reverse);

    		Stringrow
    $justify

    The justify-content property (default is center);

    		Stringcenter
    $align

    The align-items property (default is center);

    		Stringcenter

    Example

    -> in scss:
.main {
  @include flexCenter(column, flex-start, center)
} 


-> in css:
.main {
  display: flex;
  justify-content: flex-start;
  align-items: center;
  flex-direction: column;
}
